Precourse Instruction

Welcome to the Refactory Preparation Season. Preparation items are splitted to 3 main areas: Mobile Programming, Backend Programming, and DevOps. Some items can also be applied to all main areas. So, complete items based on your interest.

All progress can be upload through menu Assignment > Overview. Drag to Doing when you will work on a task. And drag to Ready when you are ready to upload your progress.

Installing Development tools

Estimation : 1 day

All applicant MUST install these development tools

  • OS Ubuntu latest (or your preferred OS. Just remember you must use Nix based OS)
  • Git
  • Node JS
  • Docker
    Ref : https://docs.docker.com/engine/installation/linux/ubuntu/#install-using-the-repository
  • Text Editor like Visual Studio Code or Atom Editor

Mobile Programming:

  • Android SDK + Android Studio
  • Genymotion (optional, you can use AVD instead)
    Ref: https://www.genymotion.com/fun-zone/

Backend Programming:

  • LAMP Stack : Apache2, MySQL, and PHP (Install it separately, don't use XAMPP)
    Ref : https://www.digitalocean.com/community/tutorials/how-to-install-linux-apache-mysql-php-lamp-stack-on-ubuntu-16-04
  • Composer
    Ref: https://getcomposer.org/doc/00-intro.md
  • Python & Flask
    Ref: http://flask.pocoo.org/docs/0.12/installation/

DevOps:

  • Kubernetes (Kubctl + Minikube)
    Ref: https://kubernetes.io/docs/getting-started-guides/minikube/
